CSS TUTORIAL

¿Qué es CSS?

  • CSS significa hojas de estilo en cascada.
  • CSS describe cómo se mostrarán los elementos HTML en la pantalla, en papel o en otros medios.
  • CSS ahorra mucho trabajo. Puede controlar el diseño de varias páginas web a la vez.
  • Las hojas de estilo externas se almacenan en archivos CSS.

¿Por qué se utiliza CSS?

CSS se utiliza para definir estilos para sus páginas web, incluido: el diseño, el diseño y las variaciones en la visualización de diferentes dispositivos y tamaños de pantalla.
